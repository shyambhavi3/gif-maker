GIF Maker
  • Spinning Wheel
  • Dice Roll
  • Coin Flip

For this project, I have used three concepts taught in ISTA 301. These three concepts are: Algorithmic Art, Generative Art, and GIF Making. I will take various inputs from the user and, at the end of the page, a GIF will be created based on this. By default, the GIF displays an ellipse rotating on the canvas with a black background. These update as the user gives input. 

  1. The first step is to spin the wheel. The shape that the wheel lands on will be in the GIF.
  2. Throw the three dice. The result of the 3 dice will be used to select the background color for the GIF. The RGB value for the background is calculated using this formula:
    let r =(dice1*50);
    let g =(dice2*50);
    let b =(dice3*50);
    background( r,g,b);
  3. Flip the coin! If it lands on the red side, then the animation in the GIF will be "A rotating shape", otherwise it will be "A shape entering and leaving the screen."
